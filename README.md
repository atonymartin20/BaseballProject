# Baseball Project

This is a project I have wanted to build for a while.  The point of this project is to grab data from multiple baseball websites and aggregate them into one site.

## Tech Stack
### Frontend
* React
* ContextAPI

### Backend
* Knex
* Node.js
* PostgresQL
